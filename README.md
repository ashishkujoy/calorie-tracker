# Calorie Tracker
Calorie Tracker is a simple web application that allows users to track their daily calorie intake and monitor their diet. The application provides an easy-to-use interface for logging meals, calculating total calories consumed.

## Features
- User authentication (sign up, log in, log out)
- Upload a meal photo and automatically extract calorie information using image recognition.
- Manual entry of meals and calorie information.
- View daily, weekly, and monthly calorie intake statistics.

## Technologies Used
- Frontend: React.js
- Backend: Node.js, Hono, Langchain, Ollama
- Database: MongoDB
- Image Recognition: Ollama Vision Model (gemma)
